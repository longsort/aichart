# Fulink Pro 최종본 – 오늘 개편 액션 묶음(복붙 불필요)

이 폴더(fulink_final)는 **오늘 아침부터 작업한 모든 개편을 이미 하나로 합친 ‘기준본’**입니다.
따라서 **순서대로 복붙/패치할 필요 없습니다.**

---

## 0. 적용 원칙
- **기준본은 이 ZIP 하나**(fulink_final)만 씁니다.
- 기존 파일에 ‘조각 패치 ZIP’를 순서대로 덮는 방식은 **충돌/누락/중복(AppSettings 등)** 위험이 큽니다.
- 앞으로는 **이 기준본에서만 수정**합니다.

---

## 1. 오늘 통합된 주요 개편(한 번에 포함)

### A) UI/UX (미니차트만 봐도 판단)
- 미니차트 상단 + 아래에 **한방 행동결정 영역**
- **확정(B/S) / 진입가능 / 관망 / 금지(LOCK, NO-TRADE)** 상태 고정
- 미니차트 아래: **강력 브리핑(행동 지시형) 카드**
- 확정 시: **Entry/SL/TP 요약 + 수수료 포함 손익/포지션/레버리지 계산 표시**

### B) 신호/리스크 엔진
- 5% 리스크 고정 모델
- 레버리지/증거금 계산
- 확정 남발 방지(조건 컷/데이터 LIVE 컷)
- NO-TRADE 자동 잠금 + 사유 표시
- AI 신뢰도(★) 표시

### C) CSV/통계(칩)
- 미니차트 아래 **네모칩 8개** (패턴/파동/레짐/유동성/다음 1·3·5봉 확률/장대 재출현 등)
- CSV 기반 확률 표시(가능한 범위)

---

## 2. 사용자가 해야 할 최소 작업(딱 3개)
1) ZIP 압축 해제 후 프로젝트 폴더로 사용
2) `flutter clean` → `flutter pub get`
3) 실행

---

## 3. CSV/과거 자동 수집(3년) 사용 흐름
- 앱 내 설정(⚙️)에서 **과거 데이터 자동 수집(3년)** 실행
- 다운로드는 분할/재시도/이어받기 방식
- 완료 후: 칩/확률/브리핑이 더 안정적으로 동작

---

## 4. 앞으로의 작업 규칙(중요)

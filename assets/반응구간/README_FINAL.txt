FULINK 최종 목적지 (Windows + Android 공용)

포함:
- 메인 통합 서식 + WHY(접힘) + 차트(접힘)
- 엔진 신호등(가격/캔들/거래/호가/큰손/로그/DB) + 타임스탬프
- AI 상태 단일화
- 진입가 1개 규칙
- 버튼 3개 실액션 + SQLite 로그
- RESULT 카드
- TP/SL/TIMEOUT 자동 결과판정 + 기본 튜닝(requiredProb)

실행:
flutter clean
flutter pub get
flutter run

Windows:
flutter run -d windows
Android:
flutter run

추가(최강패치): 상단 종가마감 보드(15m/1h/4h/1D/1W/1M) + 진행중/거의확정/확정 + 남은시간

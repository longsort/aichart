STEP16 적용(미래경로 선을 차트 위에 그리기)

1) 파일 추가
- lib/ui/widgets/future_path_painter.dart
- lib/ui/widgets/future_path_overlay.dart

2) chart_fullscreen_page.dart (또는 차트가 있는 위젯)에서,
차트 위젯을 만드는 곳을 아래처럼 감싸면 됨:

FuturePathOverlay(
  chartChild: yourChartWidget,
  fp: fp, // STEP14에서 만든 FuturePathPriceDTO
  anchorIndex: nowIndex, // 현재 캔들 인덱스
  indexToX: (i) => chartX(i), // 너 차트의 index->x 변환 함수
  priceToY: (p) => chartY(p), // 너 차트의 price->y 변환 함수
)

※ 너 차트 라이브러리에 이미 "price->pixel" API가 있으면 그대로 연결.
※ 없으면: minPrice/maxPrice와 차트 영역 높이로 선형 변환하면 됨(다음 스텝에서 제공 가능).
